RegExp.prototype.exec()方法的执行： 如果匹配成功，exec() 方法会返回一个数组（包含额外的属性 index 和 input），并更新正则表达式对象的 lastIndex属性。
                                   完全匹配成功的文本将作为返回数组的第一项，从第二项起，后续每项都对应正则表达式内捕获括号里匹配成功的文本。
                                   如果匹配失败，exec() 方法返回 null 并将 lastIndex重置为 0 。
